## Installation

### Requirements
- PHP 7.4 or higher
- MySQL 5.7 or higher
- Apache or Nginx

### Steps

1. Clone the repository
```bash
git clone https://github.com/YOUR-USERNAME/PeaceConnect.git
```

2. Create database
```bash
mysql -u root -p
CREATE DATABASE peaceconnect;
```

3. Import database schema
```bash
mysql -u root -p peaceconnect < /peaceconnect.sql
```
